Snap Links

Snap Links is a Browser Addon that lets you lasso links, checkboxes and other elements do things with them.

About Snap Links

Snap Links is a Firefox addon that lets you draw a lasso around links, checkboxes and other on-screen elements and open them in new tabs as well as other actions:

  • Open in New Tabs
  • Open in New Windows
  • Open Tabs in New Window
  • Bookmark All
  • Download All
  • Copy to Clipboard

Status

Snap Links is being re-written from the ground up to be a Web Extension.

The reasons behind this are simple, electrolysis is coming, XUL/XPCOM are on the way out and chrome style plugins are the new standard, which also happens to be both restartless and compatible with chrome, opera and in the future, other browsers.

I have put off fixing Snap Links because I have not had the time, nor desire to rewrite a large project, however, with v3 we are starting anew with no legacy code. This means some features may no longer exist while other new features are written.

I wholly encourage forking this project and submitting code and patches for inclusion.

It is doubtful that all v2 features will be rewritten for v3 without the help of others.

Great progress has been made with v3, the next release due soon will have the majority of the v2 features, but without a configuration panel. The next release will have configurable options.

Please see Dev Notes for the latest development status.
